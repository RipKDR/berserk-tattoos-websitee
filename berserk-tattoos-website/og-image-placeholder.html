OG Image Creation Instructions

This is a visual template for the og-image.jpg file. To create the actual image:

  1. Take a screenshot of the black rectangle above (1200x630px)
  2. Or use this as inspiration in Canva/Photoshop
  3. Save as og-image.jpg in the root directory
  4. Compress to under 300KB using TinyPNG.com
  5. Test at: opengraph.xyz

Alternative: Use a high-quality photo of the studio or best tattoo work as background with the "BERSERK TATTOOS" text overlaid.
